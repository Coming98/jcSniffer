# jcSniffer

基础：实现抓包，实现数据包的抓取与常见协议的识别与协议的过滤功能

挑战：实现分析，实现流追踪、协议的丢包率分析、监控、实现按进程追踪（CommView）

## 基础

### 图形化界面

- [ ]  多线程编程：生产者与消费者需要体现出来
